# 🛡️ Sentinel Guard Scheduler (2소대 경계작전 자동 근무표 생성 시스템)

본 프로젝트는 **2소대 경계작전 근무표를 자동으로 생성**하고,  
**개별 병사 통계 · 근무 공정성 분석 · 신병 투입 자동 배치 · 수동 수정 기능**을 제공하는  
**Palantir AIP 스타일의 웹 기반 근무 자동화 시스템**입니다.

Streamlit UI 기반이며, HuggingFace Spaces에서 누구나 접속 가능한 웹 형태로 동작합니다.

---

## 🚀 주요 기능

### ✔️ 1. 월간 근무표 자동 생성  
ORTools CP-SAT Solver 기반 알고리즘으로 아래 규칙을 모두 반영한 상태로 근무표 생성:

- 주/야간 정원 충족
- K15 사수끼리 같은 조 편성 금지
- 사수/부사수 역할 자동 배치
- 휴가/교육 자동 반영
- `야 → 주` 전환 금지
- `야 → 예비 → 야` 금지
- 예비 3연속 금지
- 최대 근무 일수 제한
- 팀 라벨 자동 회전 (C-B-A)
- 공정성 점수 기반 보정

---

### ✔️ 2. 개별 병사 통계 페이지  
- 주간/야간/예비/휴가 횟수 자동 집계  
- 개인별 공정성 점수  
- 공정성 차트 시각화(Bar Chart)

---

### ✔️ 3. 신병 투입 자동 재배치  
신병이 투입될 경우:

- 기존 팀 조정 자동 수행  
- 불균형 발생 시 즉시 재계산

---

### ✔️ 4. 수동 근무표 수정 UI  
- 생성된 근무표를 손쉽게 수정  
- 규칙 위반 즉시 감지  
- 수정 후 다시 자동 보정 기능 지원

---

### ✔️ 5. 완전한 파일 입출력 지원  
- 분대편성표.xlsx  
- 휴가.xlsx  
- 총기.xlsx  
- 짬표.xlsx  
- (옵션) ML CSV 힌트 파일  

모두 HuggingFace Spaces 상에서 즉시 로드됨.
